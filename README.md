# Klient Websockets





## Potrzebne oprogramowanie

- Node.js





## Uruchomienie aplikacji

- Aplikacja dziala na `localhost:3000`
- Należy sklonować repozytorium
- Przejść do głównego katalogu aplikacji
- W terminalu wpisać `npm install`, a następnie `npm start`
